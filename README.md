# Riots-Detection_Project

Rivot – AI-Based Riots Detection System

Project Description

Rivot is an AI-based system developed to detect potential riots or harmful activities from online content such as social media posts or news data.
The system analyzes text using machine learning techniques and identifies whether the content indicates a possible riot or threat.

Objectives

- To detect riot-related or harmful content using AI
- To improve public safety through early detection
- To demonstrate the use of machine learning in real-world problems
- To build a working web-based detection system

Technologies Used

- Programming Language: Python
- Framework: Flask
- Frontend: HTML, CSS, JavaScript
- Machine Learning: Scikit-learn
- Tools: GitHub, VS Code

Features

- User can enter text or data
- System analyzes the content
- Displays detection result (Safe / Alert)
- Simple and user-friendly interface
